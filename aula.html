🧠 Explicações sobre os codigos

Logo abaixo vai ter algumas explicações sobre os codigos "JavaSCript", "Codigos promts" e etc.

🖥️​ Prompt

Prompt: No JavaScript, o prompt() é uma função que exibe uma caixa de diálogo na tela, pedindo para o usuário digitar um texto ou valor. Ele pega essa resposta e a envia para o código processar. No mundo de Inteligência Artificial, "prompt" também se refere aos comandos de texto que você dá para as IAs (como o GitHub Copilot ou o ChatGPT) gerarem códigos automaticamente.

​👨🏻‍💻​ Codigos JavaScript

Códigos JavaScript: É o cérebro que dá vida à internet. Ele permite criar animações, validar formulários, construir jogos, desenvolver aplicativos móveis (via React Native) e criar servidores (com Node.js). Praticamente tudo que é dinâmico em uma página web depende dele.

💾​ APP VS Code

VS Code (Visual Studio Code): É um dos editores de código mais populares do mundo. Nele, você escreve o JavaScript e usa extensões (como o Code Runner) para testar e rodar seu programa diretamente. Ele avisa onde há erros de digitação, ajuda a organizar grandes projetos e facilita a integração com sistemas como o Git.

📋 Informações abaixo

📋 Para que usamos o VS Code?

Usamos o VS Code (Visual Studio Code) como um editor de código-fonte versátil para escrever, editar, testar e depurar programas. Ele permite trabalhar com diversas linguagens como Python, JavaScript e C++, aumentando a produtividade com ferramentas como autocompletar, terminal integrado e controle de versão.

📋 Para que usamos o JavaScript?

O JavaScript é a principal linguagem de programação usada para criar interatividade e dinamismo na internet. Enquanto o HTML cuida da estrutura e o CSS da aparência, o JavaScript é responsável pela "inteligência" de um site, permitindo exibir animações, validar formulários, alterar conteúdos sem recarregar a página e muito mais.

📋 Para que usamos o Prompt

Usamos um prompt para dar instruções ou fazer perguntas a uma Inteligência Artificial (como o ChatGPT, Gemini, Copilot) ou a um sistema de computador.
